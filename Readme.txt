             ZenDesktop Premium One-Key Deploy Package v2.5.1
========================================================================

欢迎使用 ZenDesktop 极简桌面美化套件！本套件包含以下三个本地插件：
1. ZenDesktop: Taskbar Acrylic Styler (精细化亚克力任务栏，支持多档透明度调节)
2. ZenDesktop: Start Menu Acrylic Styler (精细化亚克力开始菜单，支持多档透明度调节)
3. ZenDesktop: Double Click to Hide Icons (双击桌面空白处智能隐藏/显示图标)

========================================================================
安装与部署指南（仅需 3 步）：
========================================================================

第一步：安装 Windhawk
   双击运行 【windhawk_setup_offline.exe】 进行安装。
   此安装包为离线版，包含所有依赖，无需联网。

第二步：一键部署配置
   鼠标右键点击 【deploy.bat】，选择【以管理员身份运行】。
   脚本会自动导入全部 3 个本地 Mod 并完成注册配置。

第三步：激活 Mod
   打开 Windhawk 软件，会看到 3 个本地插件正在后台自动编译。
   等待约 60 秒后即可生效，无需额外操作。

   如需切换主题风格，在插件的 Theme 下拉菜单中挑选即可。
