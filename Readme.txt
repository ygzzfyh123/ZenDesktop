欢迎使用 ZenDesktop 极简桌面美化套件！本套件包含以下三个高能本地插件：
1. ZenDesktop: Taskbar Acrylic Styler (精细化亚克力任务栏，支持多档透明度调节)
2. ZenDesktop: Start Menu Acrylic Styler (精细化亚克力开始菜单，支持多档透明度调节)
3. ZenDesktop: Double Click to Hide Icons (双击桌面空白处智能隐藏/显示图标)

------------------------------------------------------------------------
安装与部署指南（仅需 3 步）：
------------------------------------------------------------------------

第一步：安装 Windhawk
   运行本文件夹下的 Windhawk 安装包（例如 windhawk_setup.exe）进行安装。
   （如果您的电脑已安装 Windhawk，可直接跳到第二步）

第二步：一键部署配置
   鼠标右键点击本文件夹下的 【deploy.bat】，选择【以管理员身份运行】。
   脚本会自动导入本套件的全部 3 个极简 Mod 并完成底层注册。

第三步：激活 Mod 并选主题
   1. 打开 Windhawk 软件，您会在主界面看到新增的 3 个本地插件。
   2. 分别点击 Taskbar 和 Start Menu 插件，进入后点击右上角的【保存并编译】（只需几秒即可完成本地极速编译）。
   3. 编译完成后，在设置选项 of Theme 下拉菜单中，即可自由挑选您喜欢的高级透明度预设！
